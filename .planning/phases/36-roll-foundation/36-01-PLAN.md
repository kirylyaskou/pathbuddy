Create `engine/dice/dice.ts` with the following exports (per D-01, D-02, D-03, D-04):

**Types:**

```typescript
export interface DiceEntry {
  sides: number   // e.g. 6, 20 — plain number, NOT DieFace
  value: number   // rolled result, 1..sides
}

export interface Roll {
  id: string                  // crypto.randomUUID()
  formula: string             // original formula, e.g. "2d6+4"
  dice: DiceEntry[]           // each individual die rolled
  modifier: number            // sum of all flat +N/-N terms
  total: number               // sum of all dice values + modifier
  label?: string              // e.g. "Fireball damage", "Longsword attack"
  timestamp: number           // Date.now()
}

export interface ParsedFormula {
  dice: Array<{ count: number; sides: number }>   // e.g. [{count:2, sides:6}]
  modifier: number                                  // flat sum of +N/-N
}
```

**parseFormula(formula: string): ParsedFormula**
1. Normalize: `formula.trim().replace(/\s+/g, '')`
2. Handle empty/falsy → return `{ dice: [], modifier: 0 }`
3. Tokenize with regex: `const TOKEN_RE = /([+-]?\d*d\d+|[+-]?\d+)/gi`
4. For each token, test `/^([+-]?\d*)d(\d+)$/i`:
   - Match → dice term. Count: `countStr ? parseInt(countStr) : 1` (handles bare `d20`). Handle `+d6` → count=1, `-d6` is invalid (ignore sign on dice count, treat as 1).
   - No match → flat modifier. `modifier += parseInt(token)`
5. Return `{ dice: [...], modifier }`

Edge cases per D-03:
- `"d20"` → `{dice:[{count:1, sides:20}], modifier:0}`
- `"+5"` → `{dice:[], modifier:5}`
- `"-2"` → `{dice:[], modifier:-2}`
- `"0"` → `{dice:[], modifier:0}`
- `"1d6-2"` → `{dice:[{count:1, sides:6}], modifier:-2}` (TOKEN_RE captures the `-2` with sign)
- `"2d6+1d4+3"` → `{dice:[{count:2, sides:6},{count:1, sides:4}], modifier:3}`

**rollDice(formula: string, label?: string): Roll**
1. Call `parseFormula(formula)`
2. For each dice group `{count, sides}`, generate `count` random values: `Math.floor(Math.random() * sides) + 1`
3. Collect all into `DiceEntry[]`
4. `total = sum(dice.map(d => d.value)) + modifier`
5. Return `{ id: crypto.randomUUID(), formula, dice, modifier, total, label, timestamp: Date.now() }`

**heightenFormula(base: string, heightened: { perRanks: number; add: string }, castRank: number, baseRank: number): string** (per D-04)
1. `if (castRank <= baseRank) return base`
2. `const increments = Math.floor((castRank - baseRank) / heightened.perRanks)`
3. `if (increments <= 0) return base`
4. For `i = 0..increments-1`: `base = appendFormula(base, heightened.add)` — simple string concatenation: `base + '+' + add`
5. Return the resulting formula string

The `appendFormula` helper just does string concat (`base + '+' + add`). No like-dice combining — per Research open question #2, append approach is simpler and rollDice() handles the resulting formula correctly. E.g. `heightenFormula("6d6", {perRanks:1, add:"2d6"}, 5, 3)` → `"6d6+2d6"` (which rollDice parses correctly as 8 dice of d6).

```typescript
import { create } from 'zustand'
import { immer } from 'zustand/middleware/immer'
import type { Roll } from '@engine'

interface RollStoreState {
  // Roll history
  rolls: Roll[]
  addRoll: (roll: Roll) => void
  clearRolls: () => void

  // MAP tracking (attack count per combatant per round)
  attackCountByCombatant: Record<string, number>
  incrementAttackCount: (combatantId: string) => void
  resetAttackCount: (combatantId: string) => void
  resetAllMAP: () => void
}

export const useRollStore = create<RollStoreState>()(
  immer((set) => ({
    rolls: [],
    addRoll: (roll) =>
      set((state) => {
        state.rolls.push(roll)
      }),
    clearRolls: () =>
      set((state) => {
        state.rolls = []
      }),

    attackCountByCombatant: {},
    incrementAttackCount: (combatantId) =>
      set((state) => {
        state.attackCountByCombatant[combatantId] =
          (state.attackCountByCombatant[combatantId] ?? 0) + 1
      }),
    resetAttackCount: (combatantId) =>
      set((state) => {
        delete state.attackCountByCombatant[combatantId]
      }),
    resetAllMAP: () =>
      set((state) => {
        state.attackCountByCombatant = {}
      }),
  }))
)
```

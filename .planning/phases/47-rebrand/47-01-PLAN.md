# Plan 47-01: Rebrand Pathbuddy → PathMaid

## Objective

Replace every user-visible and config-level "PathBuddy"/"pathbuddy" string with "PathMaid"/"pathmaid" across all config files, Rust sources, and React UI components. Purely mechanical text replacement — no architecture or logic changes.

## must_haves

- [ ] App window title shows "PathMaid" (tauri.conf.json `windows[0].title`)
- [ ] `productName` in tauri.conf.json is "PathMaid"
- [ ] `identifier` in tauri.conf.json is "com.pathmaid.app"
- [ ] `package.json` `name` is "pathmaid"
- [ ] Cargo.toml crate name is "pathmaid", lib name is "pathmaid_lib"
- [ ] `main.rs` calls `pathmaid_lib::run()`
- [ ] SQLite DB filename is `sqlite:pathmaid.db`
- [ ] SplashScreen shows "PathMaid"
- [ ] AppSidebar shows "PathMaid"
- [ ] sync.rs user-agent is "pathmaid/1.1.0"

**Constants:**
- `STAGES = ['checking', 'downloading', 'extracting', 'importing', 'cleanup', 'done'] as const`
- Stage display labels map: `{ checking: 'Checking for updates...', downloading: 'Downloading...', extracting: 'Extracting...', importing: 'Importing...', cleanup: 'Cleaning up...', done: 'Done!' }`

**Computed:**
- `progressPercent = computed(() => { if (!progress.value || progress.value.stage !== 'importing') return 0; const { current, total } = progress.value; if (!current || !total) return 0; return Math.min(100, Math.round((current / total) * 100)) })`
- `isDeterminate = computed(() => progress.value?.stage === 'importing')`

**onMounted:** Read syncState from DB:
```typescript
onMounted(async () => {
  const rows = await db.select().from(syncState).limit(1)
  syncStateData.value = rows[0] ?? null
})
```

**startSync function:**
```typescript
async function startSync() {
  isSyncing.value = true
  completedStages.value = []
  progress.value = null
  error.value = null
  errorDetails.value = null
  result.value = null
  showErrorDetails.value = false

  try {
    const syncResult = await syncPacks((p: SyncProgress) => {
      if (progress.value && progress.value.stage !== p.stage) {
        completedStages.value = [...completedStages.value, progress.value.stage]
      }
      progress.value = p
    })
    result.value = syncResult
    // Refresh version display from DB
    const rows = await db.select().from(syncState).limit(1)
    syncStateData.value = rows[0] ?? null
    // After 3 seconds, clear result summary to settle to version display
    setTimeout(() => { result.value = null }, 3000)
  } catch (err) {
    error.value = err instanceof Error ? err.message : String(err)
    errorDetails.value = err instanceof Error ? (err.stack ?? null) : null
  } finally {
    isSyncing.value = false
  }
}
```

**retrySync function:** Just calls `startSync()`.

**Template structure (Tailwind classes, bg-white rounded-lg shadow-md p-4 tile matching existing dashboard style):**

1. **Header:** `<h2 class="text-xl font-bold text-gray-800">Sync Data</h2>`

2. **Idle state (v-if="!isSyncing && !error && !result"):**
   - If `syncStateData` exists: show release tag (`syncStateData.lastRelease`), last synced date (formatted from `syncStateData.lastSyncAt`), entity count (`syncStateData.totalEntities?.toLocaleString()`)
   - If `syncStateData` is null: show `<p class="text-sm text-gray-500 mt-2">No PF2e data yet — sync to get started</p>`

3. **Active sync state (v-if="isSyncing"):**
   - Stage pipeline list: `v-for="stage in STAGES"` — completed stages get green checkmark `✓`, current stage gets blue animated dot, future stages get gray dot
   - Current stage message: `progress.message` (shows importing count like "Importing: 1200 / 28000")
   - Progress bar: `<div class="h-2 bg-gray-200 rounded-full overflow-hidden mt-2">`
     - If `isDeterminate`: `<div class="h-full bg-blue-500 transition-all duration-300" :style="{ width: progressPercent + '%' }" />`
     - Else: `<div class="h-full bg-blue-500 animate-pulse w-full" />`

4. **Result summary (v-if="result && !isSyncing"):**
   - Show: `Added: {result.added}, Updated: {result.updated}, Deleted: {result.deleted}` with green text
   - Show synced release: `result.release`

5. **Error state (v-if="error && !isSyncing"):**
   - Error message: `<p class="text-sm text-red-700 font-medium">{{ error }}</p>`
   - Show details toggle: `<button v-if="errorDetails" @click="showErrorDetails = !showErrorDetails">{{ showErrorDetails ? 'Hide details' : 'Show details' }}</button>`
   - Details: `<pre v-if="showErrorDetails && errorDetails">{{ errorDetails }}</pre>`
   - Retry button: `<button @click="retrySync" class="mt-2 text-sm bg-red-600 text-white px-3 py-1 rounded hover:bg-red-700">Retry</button>`

6. **Sync button (always visible at bottom):**
   - `<button @click="startSync" :disabled="isSyncing" class="mt-4 w-full bg-blue-600 text-white py-2 px-4 rounded hover:bg-blue-700 disabled:opacity-50 disabled:cursor-not-allowed">{{ isSyncing ? 'Syncing...' : 'Sync Now' }}</button>`
   - Hide the button when error is showing (retry button replaces it)

**CRITICAL PITFALLS TO HANDLE:**
- `done` stage from syncPacks is a valid progress emission — use the Promise return value (SyncResult) for result counts, NOT the done message
- syncPacks may short-circuit with immediate `done` if already up to date — don't assume all stages fire
- syncState table may be empty (never synced) — guard with `rows[0] ?? null`
- Always use `finally { isSyncing.value = false }` to prevent stuck disabled button

# Plan 10-04: Hazard XP in Encounter Builder

<objective>
Add hazard support to the encounter builder — a hazard add form with level and type (simple/complex), integrated into the existing encounter draft and XP calculation via engine `getHazardXp` and `calculateXP`.
</objective>

## Tasks

<task id="1">
<title>Extend encounter builder store for hazard entries</title>
<read_first>
- src/features/encounter-builder/model/store.ts (DraftCreature, selectEncounterResult, calculateXP usage)
- engine/encounter/xp.ts (calculateXP signature: creatures[], hazards: {level, type}[], partyLevel, partySize)
- engine/encounter/xp.ts (HazardType = 'simple' | 'complex', getHazardXp)
</read_first>
<action>
1. Add DraftHazard interface:
   ```ts
   export interface DraftHazard {
     instanceId: string
     name: string
     level: number
     type: HazardType  // 'simple' | 'complex'
   }
   ```
   Import `HazardType` from `@engine`.

2. Add hazard state and actions to EncounterBuilderState:
   ```ts
   draftHazards: DraftHazard[]
   addHazardToDraft: (hazard: { name: string; level: number; type: HazardType }) => void
   removeHazardFromDraft: (instanceId: string) => void
   ```

3. Implement actions:
   ```ts
   addHazardToDraft: (hazard) =>
     set((state) => {
       state.draftHazards.push({
         instanceId: `hazard-${++instanceCounter}`,
         name: hazard.name,
         level: hazard.level,
         type: hazard.type,
       })
     }),
   removeHazardFromDraft: (instanceId) =>
     set((state) => {
       state.draftHazards = state.draftHazards.filter((h) => h.instanceId !== instanceId)
     }),
   ```

4. Initialize `draftHazards: []` in the store default state.

5. Update `clearDraft` to also clear hazards:
   ```ts
   clearDraft: () =>
     set((state) => {
       state.draftCreatures = []
       state.draftHazards = []
     }),
   ```

6. Update `selectEncounterResult` to pass hazards to `calculateXP`:
   ```ts
   export function selectEncounterResult(state: EncounterBuilderState): EncounterResult {
     const levels = state.draftCreatures.map((c) => c.adjustedLevel ?? c.level)
     const hazards = state.draftHazards.map((h) => ({ level: h.level, type: h.type }))
     return calculateXP(levels, hazards, state.partyLevel, state.partySize)
   }
   ```
</action>
<acceptance_criteria>
- DraftHazard interface exists in store.ts with instanceId, name, level, type fields
- EncounterBuilderState has `draftHazards: DraftHazard[]`
- `addHazardToDraft` and `removeHazardFromDraft` actions exist
- `clearDraft` clears both draftCreatures and draftHazards
- `selectEncounterResult` passes hazards array to `calculateXP`
- HazardType imported from `@engine`
</acceptance_criteria>
</task>

<task id="2">
<title>Add hazard add form and display in EncounterCreatureList</title>
<read_first>
- src/features/encounter-builder/ui/EncounterCreatureList.tsx (current draft creature list rendering)
- src/features/encounter-builder/model/store.ts (after Task 1 modifications)
- engine/encounter/xp.ts (getHazardXp for per-hazard XP display)
</read_first>
<action>
1. In `EncounterCreatureList.tsx`, add a "Add Hazard" section below the creature list:
   ```tsx
   // Hazard add form — compact inline
   const [hazardName, setHazardName] = useState('')
   const [hazardLevel, setHazardLevel] = useState(1)
   const [hazardType, setHazardType] = useState<HazardType>('simple')
   const addHazard = useEncounterBuilderStore((s) => s.addHazardToDraft)
   const removeHazard = useEncounterBuilderStore((s) => s.removeHazardFromDraft)
   const draftHazards = useEncounterBuilderStore(useShallow((s) => s.draftHazards))
   const partyLevel = useEncounterBuilderStore((s) => s.partyLevel)
   ```

2. Render hazard entries in the list, visually distinct from creatures:
   ```tsx
   {draftHazards.map((h) => {
     const xpResult = getHazardXp(h.level, partyLevel, h.type)
     return (
       <div key={h.instanceId} className="flex items-center gap-2 px-3 py-1.5 border-b border-border/30">
         <span className="text-xs px-1.5 py-0.5 rounded bg-amber-900/30 text-amber-400 font-medium">
           {h.type === 'complex' ? 'Complex' : 'Simple'}
         </span>
         <span className="flex-1 text-sm truncate">{h.name || `Hazard Lv${h.level}`}</span>
         <span className="text-xs text-muted-foreground font-mono">Lv{h.level}</span>
         <span className="text-xs text-muted-foreground font-mono w-12 text-right">
           {xpResult.outOfRange ? '—' : `${xpResult.xp} XP`}
         </span>
         <button
           onClick={() => removeHazard(h.instanceId)}
           className="text-muted-foreground hover:text-destructive text-xs"
         >
           ×
         </button>
       </div>
     )
   })}
   ```

3. Add an inline hazard form at the bottom of the list:
   ```tsx
   <div className="p-2 border-t border-border/30 space-y-1.5">
     <p className="text-[10px] uppercase tracking-wider text-muted-foreground">Add Hazard</p>
     <div className="flex items-center gap-1.5">
       <Input
         value={hazardName}
         onChange={(e) => setHazardName(e.target.value)}
         placeholder="Hazard name"
         className="h-7 text-xs flex-1"
       />
       <Input
         type="number"
         value={hazardLevel}
         onChange={(e) => setHazardLevel(parseInt(e.target.value) || 1)}
         className="h-7 text-xs w-14"
         min={-1}
         max={25}
       />
       <select
         value={hazardType}
         onChange={(e) => setHazardType(e.target.value as HazardType)}
         className="h-7 text-xs rounded bg-secondary border border-border px-1"
       >
         <option value="simple">Simple</option>
         <option value="complex">Complex</option>
       </select>
       <Button
         size="sm"
         className="h-7 text-xs"
         onClick={() => {
           addHazard({ name: hazardName, level: hazardLevel, type: hazardType })
           setHazardName('')
         }}
       >
         Add
       </Button>
     </div>
   </div>
   ```

4. Import `getHazardXp` and `HazardType` from `@engine`.
</action>

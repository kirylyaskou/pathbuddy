**1. Create `src/widgets/roll-history/ui/RollHistoryPanel.tsx`**:

This component renders inside a Popover (shadcn) triggered by a button in AppHeader. It receives no props -- it reads directly from useRollStore.

Imports:
```typescript
import { Trash2 } from 'lucide-react'
import { Button } from '@/shared/ui/button'
import { Badge } from '@/shared/ui/badge'
import { Separator } from '@/shared/ui/separator'
import { useRollStore } from '@/shared/model/roll-store'
import type { Roll, DiceEntry } from '@engine'
```

Implementation:

**State access:**
- `const rolls = useRollStore(state => state.rolls)`
- `const clearRolls = useRollStore(state => state.clearRolls)`
- Display rolls in reverse order (newest first), capped at 50: `const displayRolls = [...rolls].reverse().slice(0, 50)`

**Header section** (flex items-center justify-between, p-3):
- Left: `<span className="text-sm font-semibold">Roll History</span>` + `<Badge variant="secondary" className="ml-2">{rolls.length === 1 ? '1 roll' : `${rolls.length} rolls`}</Badge>`
- Right: `<Button variant="ghost" size="sm" onClick={clearRolls} disabled={rolls.length === 0} aria-label="Clear roll history"><Trash2 className="h-3.5 w-3.5 mr-1" />Clear History</Button>`

**Separator** after header.

**Content area** (max-h-[240px] overflow-y-auto):
- If `rolls.length === 0`: Show empty state: `<div className="flex flex-col items-center justify-center py-8 text-muted-foreground"><p className="text-sm font-semibold">No rolls yet</p><p className="text-xs">Rolls will appear here during the session.</p></div>`
- Otherwise, map `displayRolls` to rows. Each row (flex items-center gap-2 px-3 h-8 hover:bg-muted/50 transition-colors duration-150):
  - **Timestamp**: `<span className="text-xs text-muted-foreground font-mono w-10 shrink-0">{formatTime(roll.timestamp)}</span>` where `formatTime` formats as `HH:MM` using `new Date(roll.timestamp).toLocaleTimeString([], { hour: '2-digit', minute: '2-digit', hour12: false })`
  - **Formula**: `<Badge variant="secondary" className="text-xs font-mono">{roll.formula}</Badge>`
  - **Die breakdown abbreviated**: `<span className="text-xs font-mono text-muted-foreground">{roll.dice.map(d => d.value).join('+')}</span>`
  - **Modifier** (if !== 0): `<span className="text-xs font-mono text-muted-foreground">{roll.modifier > 0 ? '+' : ''}{roll.modifier}</span>`
  - **Spacer**: `<span className="flex-1" />`
  - **Total**: `<span className="text-xs font-mono font-semibold" style={{ color: 'var(--pf-gold)' }}>= {roll.total}</span>`
  - If roll has label: show it as tooltip or small text after formula badge (optional, keep row compact)

**Panel container**: `<div className="w-80">...</div>` -- 320px width for the popover content.

# Plan 10-02: IWR Damage Preview

<objective>
Add a damage type combobox to the Damage input in HpControls. When a type is selected and the combatant's creature has matching IWR entries, display a breakdown row showing raw → immunities → weaknesses → resistances → final damage via engine `applyIWR`.
</objective>

2. In `CombatantDetail.tsx`, look up the creature's IWR data when rendering an NPC combatant. Since the combatant has `creatureRef` (creature entity id), use a lightweight approach:
   - Import `useCreatureStore` from `@/entities/creature` to access the creature entity by id.
   - If creature data is available, extract IWR and pass to HpControls:
   ```tsx
   <HpControls
     combatant={combatant}
     iwrImmunities={creatureIwr?.immunities}
     iwrWeaknesses={creatureIwr?.weaknesses}
     iwrResistances={creatureIwr?.resistances}
   />
   ```

   If creature data is not readily available from a store, use a simpler approach: store IWR on the Combatant at creation time in `BestiarySearchPanel.tsx` and `createCombatantFromCreature`. Extract IWR from `CreatureRow.raw_json` using the same parsing logic as `toCreatureStatBlockData`.

3. Create a helper in `src/entities/creature/model/mappers.ts`:
   ```ts
   export function extractIwr(row: CreatureRow): {
     immunities: string[]
     weaknesses: { type: string; value: number }[]
     resistances: { type: string; value: number }[]
   } {
     const raw = JSON.parse(row.raw_json)
     const system = raw.system || {}
     return {
       immunities: (system.attributes?.immunities || []).map((i: any) => i.type || String(i)),
       weaknesses: (system.attributes?.weaknesses || []).map((w: any) => ({
         type: w.type || String(w), value: w.value ?? 0,
       })),
       resistances: (system.attributes?.resistances || []).map((r: any) => ({
         type: r.type || String(r), value: r.value ?? 0,
       })),
     }
   }
   ```
</action>

</acceptance_criteria>

3. Compute IWR preview when damageType is set and damageInput has a value:
   ```tsx
   const iwrPreview = useMemo(() => {
     if (!damageType || !damageInput) return null
     const amount = parseInt(damageInput, 10)
     if (isNaN(amount) || amount <= 0) return null
     // Check if creature has any matching IWR
     const immunities = (props.iwrImmunities || [])
       .filter(t => DAMAGE_TYPES.includes(t as DamageType))
       .map(t => createImmunity(t as ImmunityType))
     const weaknesses = (props.iwrWeaknesses || [])
       .map(w => createWeakness(w.type as WeaknessType, w.value))
     const resistances = (props.iwrResistances || [])
       .map(r => createResistance(r.type as ResistanceType, r.value))
     if (immunities.length === 0 && weaknesses.length === 0 && resistances.length === 0) return null
     const instance: DamageInstance = { type: damageType, amount }
     return applyIWR(instance, immunities, weaknesses, resistances)
   }, [damageType, damageInput, props.iwrImmunities, props.iwrWeaknesses, props.iwrResistances])
   ```

4. Render IWR breakdown row below the Damage column (conditionally, only when preview exists):
   ```tsx
   {iwrPreview && (
     <div className="col-span-3 text-xs space-y-0.5 px-1 py-1.5 rounded bg-secondary/30 border border-border/30">
       <div className="flex justify-between">
         <span className="text-muted-foreground">Raw</span>
         <span className="font-mono">{damageInput}</span>
       </div>
       {iwrPreview.appliedImmunities.length > 0 && (
         <div className="flex justify-between text-blue-400">
           <span>Immune ({iwrPreview.appliedImmunities.map(i => i.type).join(', ')})</span>
           <span className="font-mono">→ 0</span>
         </div>
       )}

         <div className="flex justify-between text-red-400">
           <span>Weakness ({iwrPreview.appliedWeaknesses.map(w => `${w.type} ${w.value}`).join(', ')})</span>
           <span className="font-mono">+{iwrPreview.appliedWeaknesses[0].value}</span>
         </div>
       )}
       {iwrPreview.appliedResistances.length > 0 && (
         <div className="flex justify-between text-green-400">
           <span>Resist ({iwrPreview.appliedResistances.map(r => `${r.type} ${r.value}`).join(', ')})</span>

         </div>
       )}
       <div className="flex justify-between font-bold border-t border-border/30 pt-0.5">
         <span>Final</span>
         <span className="font-mono">{iwrPreview.finalDamage}</span>
       </div>
     </div>
   )}
   ```

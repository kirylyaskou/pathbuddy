**Status:** ✅ SHIPPED 2026-04-23 (retroactive archive, 2026-04-23)

## Key Decisions

- **Desktop-only (Windows + Linux)** — Android удалён из CI. macOS: shell.open на GitHub Releases (notarization отложена).
- **ed25519 signing** — встроено в tauri-plugin-updater; private key в GitHub Secrets.
- **NSIS canonical Windows format** — `updaterJsonPreferNsis: true`. Не .msi.
- **Custom UpdateDialog UI** — Tauri v2 удалил native dialog, custom UI обязателен.
- **`db.close()` до install()** — без этого NSIS падает с "Failed to kill" на Windows.
- **Parallel-safe Phase 74/75** — через отдельные git worktree, слияние после typecheck.
- **shared/api/updater.ts — единственный importer** `@tauri-apps/plugin-updater` (FSD IPC boundary).

## Technical Debt Incurred

- **macOS update flow** — только shell.open, full в-app updater отложен до notarization (future milestone).
- **latest.json.platforms structure** — hand-rolled в `main.yml`, уязвимо к API breaking changes в tauri-plugin-updater.
- **UAT manually только Windows** — Linux cross-device `os error 18` проверен pre-release, но без CI E2E.

**D. CombatPage.tsx — CombatColumn component + split rendering**

1. Create a `CombatColumn` component inside CombatPage.tsx (or as a separate file, but inline is simpler for this scope). CombatColumn receives a tab and renders a self-contained initiative list + detail + controls.

Key architectural decision (per RESEARCH.md recommendation): In split mode, each column needs to operate on its tab's data. The simplest approach is: when a user interacts with a split column (clicks a combatant, clicks Start/Next), first call `setActiveTab(tab.id)` to make that tab active (which saves the other tab's snapshot and restores the clicked tab's data to global stores). Then the existing components (InitiativeList, CombatantDetail, CombatControls, TurnControls) work as-is because they read from global stores.

The visual trick: both columns render simultaneously, but only the "focused" column's data is live in global stores. The other column renders from its snapshot. To achieve this without rewriting all widgets:
- Each column renders the standard widgets (InitiativeList, CombatantDetail, CombatControls, TurnControls)
- Only the active column's widgets are connected to global stores
- The inactive column renders a read-only snapshot view

Simplification: Given the complexity of making both columns truly independent (would require rewriting all widgets to accept props), use a simpler approach:
- Split mode renders both columns visually
- Each column has its own `selectedId` state
- Clicking anywhere in a column calls `setActiveTab(column.tab.id)` first
- The active column renders the full interactive widgets
- The inactive column renders a lightweight read-only view from its snapshot

Create `CombatColumn`:
```typescript
interface CombatColumnProps {
  tab: EncounterTab
  isActive: boolean
  onActivate: () => void
  onSelect: (id: string) => void
  className?: string
}

function CombatColumn({ tab, isActive, onActivate, onSelect, className }: CombatColumnProps) {
  const [columnSelectedId, setColumnSelectedId] = useState<string | null>(null)

  const handleColumnSelect = useCallback((id: string) => {
    onActivate() // setActiveTab to this column's tab
    setColumnSelectedId(id)
    onSelect(id) // triggers stat block load in parent
  }, [onActivate, onSelect])

  if (isActive) {
    // Active column — render full interactive widgets (they read from global stores which are synced to this tab)
    return (
      <div className={cn('flex flex-col h-full border-t-2 border-t-primary', className)} onClick={onActivate}>
        <div className="flex items-stretch shrink-0">
          <div className="flex-1"><CombatControls /></div>
          <div className="flex items-center px-2 border-b border-border/50"><AddPCDialog /></div>
        </div>
        <ResizablePanelGroup direction="vertical" className="flex-1">
          <ResizablePanel defaultSize={35} minSize={20}>
            <InitiativeList selectedId={columnSelectedId} onSelect={handleColumnSelect} />
          </ResizablePanel>
          <ResizableHandle withHandle />
          <ResizablePanel defaultSize={65} minSize={30}>
            <div className="flex flex-col h-full">
              {columnSelectedId ? (
                <div className="flex-1 min-h-0">
                  <CombatantDetail combatantId={columnSelectedId} />
                </div>
              ) : (
                <div className="flex-1 flex items-center justify-center text-muted-foreground">
                  <p className="text-sm">Select a combatant</p>
                </div>
              )}
              <TurnControls />
            </div>
          </ResizablePanel>
        </ResizablePanelGroup>
      </div>
    )
  }

  // Inactive column — render read-only snapshot view
  const combatants = tab.snapshot.combatants
  return (
    <div className={cn('flex flex-col h-full opacity-80', className)} onClick={onActivate}>
      {/* Minimal header showing round/state */}
      <div className="px-3 py-1.5 border-b border-border/50 text-xs text-muted-foreground shrink-0">
        {tab.snapshot.isRunning ? `Round ${tab.snapshot.round}` : 'Not started'}
      </div>
      {/* Read-only initiative list from snapshot */}
      <ScrollArea className="flex-1">
        <div className="p-2 space-y-0.5">
          {combatants.map((c) => (
            <div
              key={c.id}
              className={cn(
                'flex items-center gap-2 px-2 py-1 rounded text-sm cursor-pointer hover:bg-secondary/30',
                c.id === tab.snapshot.activeCombatantId && 'bg-primary/10 border-l-2 border-primary',
                c.id === columnSelectedId && 'bg-secondary/40'
              )}
              onClick={(e) => {
                e.stopPropagation()
                onActivate()
                setColumnSelectedId(c.id)
                onSelect(c.id)
              }}
            >
              <span className="text-xs font-mono text-muted-foreground w-6 text-right">{c.initiative}</span>
              <span className="flex-1 truncate">{c.displayName}</span>
              <span className="text-xs text-muted-foreground">{c.hp}/{c.maxHp}</span>
            </div>
          ))}
        </div>
      </ScrollArea>
    </div>
  )
}
```

<action>
Replace the entire content of `src/shared/ui/scroll-area.tsx` with:

```tsx
import * as React from 'react'
import { cn } from '@/shared/lib/utils'

function ScrollArea({
  className,
  children,
  ...props
}: React.HTMLAttributes<HTMLDivElement>) {
  return (
    <div
      data-slot="scroll-area"
      className={cn('relative overflow-y-auto scrollbar-thin', className)}
      {...props}
    >
      {children}
    </div>
  )
}

function ScrollBar(_props: Record<string, unknown>) {
  return null
}

export { ScrollArea, ScrollBar }
```

Key changes:
- Remove all `@radix-ui/react-scroll-area` imports
- `ScrollArea` becomes a plain `<div>` with `overflow-y-auto` and `scrollbar-thin` class
- `ScrollBar` becomes a no-op returning `null` — no consumer renders it directly
- `data-slot="scroll-area"` preserved for any CSS selectors or test queries
- Props type changed from Radix-specific to `React.HTMLAttributes<HTMLDivElement>`

The `.scrollbar-thin` CSS class is added by Plan 11-01 Task 1 in globals.css, providing:
- `scrollbar-width: thin` (Firefox/standard)
- `scrollbar-color: oklch(0.30 0.04 55) transparent` (matches --border dark-mode token)
- WebKit pseudo-elements for Tauri's WebView

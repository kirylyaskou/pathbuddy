## Color

All values reference CSS custom properties from `globals.css` (Golden Parchment theme). Never hardcode OKLCH values.

| Role | Token | Usage |
|------|-------|-------|
| Dominant (60%) | `bg-background` | Page background |
| Secondary (30%) | `bg-secondary/30` → `bg-secondary/50` on hover | PC card background |
| Card border | `border-border/40` → `border-border/70` on hover | PC card border |
| Primary accent | `text-primary` / `bg-primary` | Import button, active states |
| Destructive | `bg-destructive text-destructive-foreground` | Delete AlertDialog action button |
| Muted text | `text-muted-foreground` | "Class • Level" and ancestry lines on cards |
| Error text | `text-destructive text-xs` | Inline validation errors in Import Dialog |
| Empty state text | `text-muted-foreground` | Empty state heading and subtitle |

Accent reserved for: Import Character button (primary CTA), hover border on hovered card only.

### Import Dialog

```
<Dialog open={open} onOpenChange={setOpen}>
  <DialogContent className="sm:max-w-md">
    <DialogHeader>
      <DialogTitle>Import Character</DialogTitle>
    </DialogHeader>
    <Tabs defaultValue="file">
      <TabsList className="w-full">
        <TabsTrigger value="file" className="flex-1">File</TabsTrigger>
        <TabsTrigger value="paste" className="flex-1">Paste JSON</TabsTrigger>
      </TabsList>
      <TabsContent value="file">
        {/* Drag-and-drop zone */}
        <div className="border-2 border-dashed border-border/50 rounded-md p-8 text-center
                        hover:border-border/80 transition-colors cursor-pointer"
             onDragOver={...} onDrop={...} onClick={triggerFilePicker}>
          <Upload className="w-8 h-8 mx-auto mb-2 text-muted-foreground/50" />
          <p className="text-sm text-muted-foreground">Drop your Pathbuilder export here</p>
          <p className="text-xs text-muted-foreground/60 mt-1">or click to browse</p>
        </div>
        <input type="file" accept=".json" className="hidden" ref={fileInputRef} onChange={onFileChange} />
        {error && <p className="text-xs text-destructive mt-2">{error}</p>}
      </TabsContent>
      <TabsContent value="paste">
        <Textarea
          placeholder='Paste Pathbuilder JSON here...'
          className="min-h-[160px] font-mono text-xs resize-none"
          value={pasteValue} onChange={e => setPasteValue(e.target.value)}
        />
        {error && <p className="text-xs text-destructive mt-2">{error}</p>}
      </TabsContent>
    </Tabs>
    <DialogFooter>
      <Button variant="outline" onClick={() => setOpen(false)}>Close</Button>
      <Button onClick={handleImport} disabled={importing}>
        {importing ? <Loader2 className="w-4 h-4 mr-1.5 animate-spin" /> : null}
        Import Character
      </Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

Icons: `Upload` for drop zone, `Loader2` (spinning) for import in-progress state.
